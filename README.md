# ShopMock QA Training Programme

Welcome to ShopMock — a fictional e-commerce platform used to train QA engineers.

## What is ShopMock?
ShopMock is a simple online gadget store. Users can:
- Create an account and log in
- Search for products
- Add items to a cart
- Complete a checkout

The code is intentionally simple. What we care about here is **process** — how we communicate, plan, test, and report.
