# ShopMock — Application Specification

ShopMock is described here in plain English. There is no live running app — your job as QA is to test the *specification* and identify gaps, ambiguities, and potential failure points.

## Features
See the `features/` folder for individual specs:

- [Login](features/login.md)
- [Product Search](features/search.md)
- [Add to Cart](features/cart.md)
- [Checkout](features/checkout.md)

## Users
ShopMock has two types of user:
- **Guest** — can browse and search, cannot purchase
- **Registered User** — can log in, add to cart, and checkout
